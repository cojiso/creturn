# Store Descriptions (DA)

## Kort beskrivelse (maks. 132 ord)
Enter til linjeskift, Ctrl+Enter for at sende på enhver side. Brugerdefinerede konfigurationer understøttes. Forhindrer utilsigtet afsendelse under IME-konvertering.

## Fuldt beskrivelser
▍Funktioner:
- 12+ indbyggede websteder og ubegrænsede brugerdefinerede tilføjelser
- Indlæs brugerdefinerede konfigurationsfiler oprettet på GitHub
- Pålidelig forebyggelse af utilsigtet afsendelse selv i chatredigeringsfelter

▍Kernefunktionalitet:
- Enter-tast = linjeskift + bekræftelse af tegnkonvertering
- Ctrl+Enter (macOS: Command+Enter) = send besked
- Ét-klik tænd/sluk

▍Understøttede websteder:
- ChatGPT, Claude, Gemini, Perplexity, Discord, Google Meet og flere
- Fuldstændig webstedsliste (konfigurationsfil):
  https://github.com/cojiso/creturn/blob/main/public/creturn-config.jsonc

▍Brugerdefineret konfiguration:
1. Opret JSONC-webstedskonfiguration
2. Upload til GitHub
3. Registrer fil-URL i udvidelsesindstillinger

▍Problemer eller funktionsønsker: GitHub-repositorium
https://github.com/cojiso/creturn
