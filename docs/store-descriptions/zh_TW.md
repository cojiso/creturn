# Store Descriptions (ZH_TW)

## 短文說明（最多 132 字）
Enter 換行，Ctrl+Enter 在任何網站上送出。支援自訂設定。可避免輸入法轉換時誤送。

## 詳細說明
▍特色：
- 內建 12+ 站點，並可無限制新增自訂站點
- 載入在 GitHub 上建立的自訂設定檔
- 即使在聊天編輯區也能可靠防止誤送

▍核心功能：
- Enter 鍵 = 換行 + 文字轉換確認
- Ctrl+Enter（macOS：Command+Enter）= 送出訊息
- 一鍵開/關切換

▍支援的網站：
- ChatGPT、Claude、Gemini、Perplexity、Discord、Google Meet 等
- 完整站點清單（設定檔）：
  https://github.com/cojiso/creturn/blob/main/public/creturn-config.jsonc

▍自訂設定：
1. 建立 JSONC 站點設定檔
2. 上傳到 GitHub
3. 在擴充功能設定中註冊檔案 URL

▍問題或功能需求：GitHub 倉庫
https://github.com/cojiso/creturn
