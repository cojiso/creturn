# Store Descriptions (RU)

## Краткое описание (до 132 слов)
Enter для переносов строки, Ctrl+Enter для отправки на любом сайте. Поддерживаются пользовательские конфигурации. Предотвращает случайную отправку во время преобразования IME.

## Полное описание
▍Особенности:
- 12+ встроенных сайтов и неограниченное добавление пользовательских сайтов
- Загрузка пользовательских конфигурационных файлов, созданных на GitHub
- Надежное предотвращение случайной отправки даже в областях редактирования чата

▍Основная функциональность:
- Enter = переносы строк + подтверждение преобразования символов
- Ctrl+Enter (macOS: Command+Enter) = отправка сообщения
- Переключатель в один клик

▍Поддерживаемые сайты:
- ChatGPT, Claude, Gemini, Perplexity, Discord, Google Meet и другие
- Полный список сайтов (файл конфигурации):
  https://github.com/cojiso/creturn/blob/main/public/creturn-config.jsonc

▍Пользовательская конфигурация:
1. Создайте JSONC-файл конфигурации сайта
2. Загрузите на GitHub
3. Зарегистрируйте URL файла в настройках расширения

▍Проблемы или запросы функций: репозиторий GitHub
https://github.com/cojiso/creturn
