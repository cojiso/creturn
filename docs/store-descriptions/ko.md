# Store Descriptions (KO)

## 짧은 설명 (최대 132단어)
Enter로 줄바꿈, Ctrl+Enter로 어떤 사이트에서든 전송. 사용자 지정 설정을 지원합니다. IME 변환 중 오발송을 방지합니다.

## 전체 설명
▍특징:
- 기본 12개 이상 사이트 + 무제한 커스텀 사이트 추가
- GitHub에 만든 커스텀 설정 파일 불러오기
- 채팅 편집 영역에서도 오발송을 안정적으로 방지

▍핵심 기능:
- Enter 키 = 줄바꿈 + 문자 변환 확정
- Ctrl+Enter (macOS: Command+Enter) = 메시지 전송
- 원클릭 온/오프 토글

▍지원 사이트:
- ChatGPT, Claude, Gemini, Perplexity, Discord, Google Meet 등
- 전체 사이트 목록(설정 파일):
  https://github.com/cojiso/creturn/blob/main/public/creturn-config.jsonc

▍커스텀 설정:
1. JSONC 사이트 설정 파일 생성
2. GitHub에 업로드
3. 확장 프로그램 설정에 파일 URL 등록

▍문제 또는 기능 요청: GitHub 저장소
https://github.com/cojiso/creturn
