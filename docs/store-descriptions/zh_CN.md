▍功能：
- 内置 12+ 站点，支持无限添加自定义站点
- 加载在 GitHub 上创建的自定义配置文件
- 即使在聊天编辑区域也能可靠防止误发送

▍核心功能：
- Enter 键 = 换行 + 文字转换确认
- Ctrl+Enter（macOS：Command+Enter）= 发送消息
- 一键开/关切换

▍支持的网站：
- ChatGPT、Claude、Gemini、Perplexity、Discord、Google Meet 等
- 完整站点列表（配置文件）：
  https://github.com/cojiso/creturn/blob/main/public/creturn-config.jsonc

▍自定义配置：
1. 创建 JSONC 站点配置文件
2. 上传到 GitHub
3. 在扩展设置中注册文件 URL
