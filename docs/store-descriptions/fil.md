# Store Descriptions (FIL)

## Maikling Paglalarawan (max 132 words)
Enter para sa bagong linya, Ctrl+Enter para magpadala sa anumang site. Sinusuportahan ang custom na mga konfigurasyon. Pinipigilan ang aksidenteng pagpapadala habang nagko-convert ang IME.

## Buong Paglalarawan
▍Mga tampok:
- 12+ built-in na site at walang limitasyong custom na dagdag
- I-load ang custom na configuration files na ginawa sa GitHub
- Maaasahang pag-iwas sa aksidenteng pagpapadala kahit sa chat editing areas

▍Pangunahing functionality:
- Enter key = bagong linya + kumpirmasyon ng character conversion
- Ctrl+Enter (macOS: Command+Enter) = magpadala ng mensahe
- One-click on/off toggle

▍Mga sinusuportahang site:
- ChatGPT, Claude, Gemini, Perplexity, Discord, Google Meet, at iba pa
- Buong listahan ng site (config file):
  https://github.com/cojiso/creturn/blob/main/public/creturn-config.jsonc

▍Custom configuration:
1. Gumawa ng JSONC site configuration file
2. I-upload sa GitHub
3. Irehistro ang file URL sa extension settings

▍Mga isyu o feature requests: GitHub repository
https://github.com/cojiso/creturn
